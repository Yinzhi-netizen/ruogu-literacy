# 若谷自主阅读计划

给若谷暑假复习一年级汉字用的 PWA 测试版。

## 当前游戏机制

- 认读关：看到一个汉字，从几个拼音里选出正确读音，完成后获得 1 星。
- 听写关：听到语音提示，例如“林，森林的林”，在 iPad 手写框里写出对应汉字，确认写对后获得 3 星。
- 阅读关：自己读完一段话或一首诗，完成后获得 5 星。
- 每 10 星触发一次铸兵动画，并点亮当前神兵的 1 个碎片。
- 金箍棒需要 3 个碎片，火尖枪需要 5 个碎片，后续神兵通常需要 10 个碎片。
- 当前神兵碎片全部点亮后，自动收入若谷的武器库。
- 神兵库展示武器出处、人物和简介。

## 当前视觉方向

- 固定为“三国铜金”配色。
- 整体定位是“汉字闯关游戏”，不是普通识字卡片。

## 后续最值得迭代

- 用若谷实际课本的识字表替换测试字库。
- 增加手写识别或家长检查模式。
- 增加更多短文和古诗。
- 给已解锁神兵增加升级、音效或展示动画。
